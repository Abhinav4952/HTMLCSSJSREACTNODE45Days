# Day 22 — ES2015 Toolkit: Spread, Rest, Destructuring, `for..of`

## Learning objectives

- Use **object/array spread** for shallow copies and overrides without manual loops.
- Collect variadic arguments with **rest parameters** (`...args`) and forward them safely.
- Destructure **arrays and objects** in parameters and assignments with **defaults** and **renames**.
- Iterate iterable structures with **`for...of`** (and combine with `Object.entries` when working with plain objects).

## Prerequisites

- **Days 17–21** (values through functions/defaults).
