# Day 20 — Arrays: Methods, Searching, and a Light `reduce`

## Learning objectives

- Use **immutable** array transforms (`filter`, `map`, `slice`) without mutating source data unintentionally.
- Sort arrays with explicit **comparator** semantics (stable expectations, case rules).
- Implement a small **reduction** that aggregates structured rows (`reduce` intro).
- Search arrays with **`findIndex` / `includes` / `indexOf`** patterns and predictable “not found” behavior.

## Prerequisites

- **Day 19** (objects/records) and **Day 18** (loops).
