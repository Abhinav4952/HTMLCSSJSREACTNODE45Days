# Day 09 — Display & Layout Patterns (No CSS Grid Required)

## Learning objectives

- Control **`display: block | inline | inline-block | flex`** and predict how whitespace, width, and alignment behave.
- Build a classic **“holy grail”**-shaped page shell using **Flexbox** (allowed) while honoring the course map note: **no CSS Grid requirement** (avoid `display: grid` in tasks unless instructor overrides).
- Diagnose the infamous **`inline-block` gap** caused by HTML whitespace and fix it intentionally (not magically).
- Implement a **media object**-style row (figure + body) using flex patterns you can reuse inside React later.

## Prerequisites

- Days **07–08** (box model + positioning), because flex items and `min-height: 0` interact with overflow (preview).
