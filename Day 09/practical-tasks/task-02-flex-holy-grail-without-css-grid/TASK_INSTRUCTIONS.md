# Task 02 — Flex “Holy Grail” (No CSS Grid)

## Summary

Implement a responsive **header + 3-column middle + footer** shell using **flexbox** only. **Do not** use `display: grid`.

## Acceptance

- Middle row grows to fill remaining viewport height under header/footer (`min-height: 100vh` shell pattern).
- Sidebars have fixed widths; main column shrinks/grows; at 720px and below, stack into a single column **without grid** (use `flex-wrap` or `flex-direction` changes via media query).
