# Day 25 — Functional Patterns: Callbacks, Chaining, Currying, Memoization

## Learning objectives

- Structure programs with **higher-order functions** and **callbacks** safely (pure-ish transforms).
- Build a small **fluent API** using method chaining (`return this`).
- Implement **currying** for binary operations.
- Apply **memoization** with a cache key strategy that avoids object key-order traps.

## Prerequisites

- **Days 20–24** (arrays/HOFs through `bind`).
