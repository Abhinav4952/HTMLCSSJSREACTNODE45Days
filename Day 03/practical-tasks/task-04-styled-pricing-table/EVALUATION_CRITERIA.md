# Evaluation Criteria — Day 03 — Task 04 — Styled Pricing Table

## Scope

Graded: semantics + visual design of pricing table.

## Weighting (100 points)

| Category | Points | Description |
|----------|-------:|-------------|
| Correctness | 45 | Valid table; scopes; aligned numbers |
| Requirements coverage | 35 | Three tiers; recommended column |
| Code quality | 20 | CSS organization; no inline styles unless TODO asks |
| Readability | 10 | Clear tier naming |

## Pass / fail gates

- Using `<table>` only for non-tabular layout.
- Fewer than three tiers.

## AI grading prompt

Verify pricing matrix, caption, thead th scopes, tbody structure, recommended column labeling and styling, tabular alignment CSS.
