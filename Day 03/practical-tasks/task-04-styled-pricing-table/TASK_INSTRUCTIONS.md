# Task 04 — Styled Pricing Table

## Summary

Build a **pricing comparison** table for three fictional tiers (labels in your own words). Style header vs body, zebra striping optional, highlight one “recommended” column.

## Learning goals

- Combine semantics from Tasks 02–03 with deliberate visual hierarchy.
- Keep numbers aligned for comparison.

## Provided files

- `index.html`, `styles/main.css`

## What you will implement

1. Table with `caption`, `thead`, `tbody`, proper `th` scopes.
2. A column (or cell in header row) marked as **Recommended** via text + styling (e.g. class on `col` via `colgroup`/`col` **or** class on each cell in that column — pick one approach documented in a brief HTML comment).
