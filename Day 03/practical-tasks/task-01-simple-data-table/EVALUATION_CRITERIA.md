# Evaluation Criteria — Day 03 — Task 01 — Simple Data Table

## Scope

Graded: `index.html`, `styles/main.css`.

## Weighting (100 points)

| Category | Points | Description |
|----------|-------:|-------------|
| Correctness | 45 | Valid table; caption; dimensions |
| Requirements coverage | 35 | TODO map satisfied |
| Code quality | 20 | Clean rows; readable labels |
| Readability | 10 | Sensible invented data |

## Pass / fail gates

- Missing `<caption>`.
- Fewer than three rows or three columns of meaningful cells.

## AI grading prompt

Verify 3×3 table, caption, first row labels in td per task spec, borders/padding in CSS.
