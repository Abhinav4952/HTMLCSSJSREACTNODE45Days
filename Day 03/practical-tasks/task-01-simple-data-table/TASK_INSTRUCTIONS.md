# Task 01 — Simple Data Table

## Summary

Create a **3×3** (minimum) table of invented but realistic data (e.g. books you read, meals, anything) with `<table>`, `<tr>`, `<td>`, and a `<caption>`.

## Learning goals

- Use `<caption>` as the accessible table title.
- Keep header row as **data cells** for this task only (`<td>` in first row) — you will upgrade to `<th>` in Task 02.
