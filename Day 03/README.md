# Day 03 — Tables

## Learning objectives

- Build **data tables** with `<table>`, `<tr>`, `<td>`, `<th>`.
- Use `<thead>`, `<tbody>`, and optional `<tfoot>` for structure.
- Associate header cells with data via `scope` (and `headers`/`id` when needed later).
- Apply `colspan` / `rowspan` safely without destroying row integrity.
- Style tables for readability while avoiding layout tables for page chrome.

## Prerequisites

- Days 01–02 (document structure, text semantics).
