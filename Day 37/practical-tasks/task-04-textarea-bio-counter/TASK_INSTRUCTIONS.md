# Task 4 — Textarea with live counter

## Summary

Practice **controlled textareas** with a **character budget** and lightweight **warning styling** as the limit approaches.

## What you will implement

1. Controlled `<textarea>` with `maxLength={140}`.
2. Live **Remaining:** counter.
3. CSS class hints: `< 20` chars remaining → `warn`; `0` remaining → `danger` on the counter span.
