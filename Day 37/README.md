# Day 37 — Forms in React (Controlled Inputs & Validation UX)

## Learning objectives

- Build **controlled** inputs where React state is the source of truth.
- Understand **uncontrolled** inputs with refs only at a high level (this day focuses on controlled patterns).
- Implement lightweight **validation UX**: messages, disabling submit, and accessible labeling.

## Prerequisites

- **Days 34–36** (props, state, composition).
