# Day 05 — CSS Selectors & Decoration

## Learning objectives

- Combine **simple selectors** (type, class, id) with **combinators** (descendant space, child `>`, adjacent sibling `+`).
- Style **borders**, **border-radius**, and **backgrounds** (`background-color`, simple `linear-gradient` optional if you keep it readable).
- Use **`text-decoration`** and **`text-transform`** intentionally (not as a hack for headings unless semantically appropriate).
