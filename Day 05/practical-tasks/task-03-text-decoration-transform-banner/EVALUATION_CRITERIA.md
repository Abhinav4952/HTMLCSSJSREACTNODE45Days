# Evaluation Criteria — Day 05 — Task 03 — Banner Typography

## Scope

Graded: typography treatments + accessible link focus.

## Weighting (100 points)

| Category | Points | Description |
|----------|-------:|-------------|
| Correctness | 45 | Kicker styling; link states |
| Requirements coverage | 35 | TODO map |
| Code quality | 20 | No inline styles |
| UX / a11y | 10 | Focus-visible present; contrast OK |

## Pass / fail gates

- Links styled with `text-decoration: none` and **no** visible focus/hover replacement.
- Body copy entirely uppercase (readability fail).

## AI grading prompt

Verify banner structure, kicker uppercase/letter-spacing, link underline variant, hover and focus-visible styles.
