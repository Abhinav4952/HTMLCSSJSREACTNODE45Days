# Task 03 — Text Decoration & Transform Banner

## Summary

Create a **marketing banner** strip: kicker line in uppercase with letter-spacing, headline, subcopy link with custom underline treatment (no `text-decoration: none` without replacement per theory).

## Learning goals

- Apply `text-transform` + `letter-spacing` tastefully.
- Combine `text-decoration` variants or border-bottom underlines for links.

## Provided files

- `index.html`, `styles/main.css`

## What you will implement

Complete TODOs in HTML for copy + CSS for styles.

## `TODO` map

| TODO | Done means |
|------|------------|
| HTML | `.banner` contains `.kicker`, `h2`, `p` with `<a href="#">` link text you write |
| CSS kicker | uppercase + letter-spacing + smaller font-size |
| CSS headline | no uppercase; comfortable line-height |
| CSS link | underline style non-default (dotted/wavy) OR border-bottom technique; include :hover and :focus-visible states |
