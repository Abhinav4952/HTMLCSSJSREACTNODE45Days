# Task 02 — Borders & Background Cards

## Summary

Style three **pricing cards** using borders, `border-radius`, and subtle **background** differences. One card must be visually “featured” using a class on the `<article>`.

## Learning goals

- Layer border + background + spacing (`padding`, `gap`) without layout frameworks.

## Provided files

- `index.html`, `styles/main.css`

## What you will implement

1. Complete HTML TODOs to fill card titles/prices with **invented** product names (not real trademarks).
2. Complete CSS TODOs for card layout in a responsive **auto-wrapping row** using `display: flex; flex-wrap: wrap; gap: ...` on `.deck`.

## `TODO` map

| TODO | Done means |
|------|------------|
| HTML | Three `<article class="card">` plus one `featured` class on middle article |
| CSS deck | Flex row wrap with gap |
| CSS card | border, radius, padding, background |
| CSS featured | distinct border-color + slightly different background |

## Manual checks

- [ ] Three cards readable at ~360px width (wrap, not overflow).
