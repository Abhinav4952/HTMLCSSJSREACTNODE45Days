# Day 01 — Basic Page Structure & Metadata

## Learning objectives

- Declare a correct HTML5 document with `<!DOCTYPE html>` and explain what breaks without it.
- Contrast HTML5 with XHTML-style habits that still appear in legacy systems and tooling.
- Structure every page with `<html>`, `<head>`, and `<body>` and know what belongs in each.
- Add essential metadata (`charset`, `viewport`, `title`) and optional `<meta>` for SEO/social previews.
- Link external CSS with `<link>` and external JS with `<script>`, choosing `defer` or `async` appropriately.

## Prerequisites

- A text editor and a modern desktop browser (Chrome, Firefox, Safari, or Edge).
- No prior course days required; this is the starting day.
