# Evaluation Criteria — Day 01 — Task 04 — Structured Profile Page

## Scope

Graded: `index.html` structure and copy; `styles/main.css` may be lightly edited only if candidate stays within task constraints (default: HTML-only edits expected).

## Weighting (100 points)

| Category | Points | Description |
|----------|-------:|-------------|
| Correctness | 35 | Valid HTML5; landmarks sensibly used; CSS loads |
| Requirements coverage | 35 | All TODOs satisfied; original bio text |
| Code quality | 20 | Clean indentation; no excessive div soup |
| Docs / readability | 10 | Readable writing; heading hierarchy sensible |

## Pass / fail gates

- Missing `<!DOCTYPE html>` or missing `charset`/`viewport`.
- Copied lorem ipsum for both sections without personalization (academic integrity gate—fail or require resubmit per policy).

## Detailed rubric

### UX / accessibility

- Single `<h1>`; sections have `<h2>`.
- Footer text legible; sufficient color contrast inherited from CSS.

## Evidence the grader should collect

- Screenshot of full page at desktop width optional.

## AI grading prompt (paste into your grader)

Evaluate Day 01 Task 04 profile page. Verify doctype, html lang, charset, viewport, title, link to styles/main.css, header/main/footer structure, one h1, two sections with h2, personalized paragraphs, footer last updated. Flag lorem ipsum-only content.
