# Day 15 — Advanced Selectors, Transitions, and Motion

## Learning objectives

- Use **attribute selectors** (`^=`, `$=`, `*=`, `=`) to style real-world variants without class explosion.
- Apply **`transform` + `transition`** for interaction feedback while avoiding layout-thrashing properties in animation paths.
- Build **`@keyframes`** animations with a **`prefers-reduced-motion`** safe fallback.
- Use **`resize`** thoughtfully for power-user panels (paired with overflow and tokens).

## Prerequisites

- Days **05–14** (selectors through flexbox). If specificity feels fuzzy, revisit **Day 11**.

## How to use this folder

1. Read `DAY-15-THEORY-AND-REFERENCE.md`.
2. Task 02 is the **transition vs layout** challenge; Task 03 requires reduced-motion handling.
