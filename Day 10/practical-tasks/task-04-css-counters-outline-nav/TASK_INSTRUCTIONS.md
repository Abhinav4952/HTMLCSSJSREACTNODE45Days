# Task 04 — **Challenge** — CSS Counters “Outline Nav”

## Summary

Build a small vertical nav whose **numbers are generated by CSS counters** (not typed manually in HTML). The HTML contains plain link text (“Foundations”, “Layout”, “Components”) but the numbers must update automatically if items are reordered.

## Rules

- You may edit `index.html` **only** to add `class` attributes or wrapper elements if needed (default: try without).
- Numbers must come from `counter-reset` / `counter-increment` / `content: counter(...)`.

## Acceptance

- Each nav item shows `1. …`, `2. …`, etc.
- Reordering the `<a>` elements in HTML updates numbering without editing CSS counters manually.
