# Task 04 — **Challenge** — Triple Collapse Clinic

## Summary

Three mini cases are stacked in one page. Each case has a “broken” outcome and a **target** described in `styles/main.css` comments. Fix **all three** with CSS only (you may edit `styles/main.css` freely; `index.html` is read-only).

## Learning goals

- Combine multiple collapse prevention techniques without fighting each other.

## Provided files

- `index.html` (read-only)
- `styles/main.css`

## Acceptance

For each case, the vertical distance between the **two highlighted borders** must match the target comment within **±1px**.
