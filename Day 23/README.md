# Day 23 — Advanced Expressions: Hoisting, Comparisons, Coercion

## Learning objectives

- Predict outcomes of **abstract equality** vs **strict equality** and know when to prefer `Object.is`.
- Recognize **truthiness** tables and use **`??`** where “only `null`/`undefined` should default.”
- Trace **numeric/string coercion** in `+`, relational operators, and `Number(...)`.
- Build small **normalization helpers** that make comparisons explicit instead of accidental.

## Prerequisites

- **Days 17–22** (values through ES2015 toolkit).
