# Task 01 — `typeof` + Primitives Lab (Vitest)

## Summary

Implement two tiny helpers used in logging and guards: **`isPrimitiveValue`** and **`typeLabel`**.

## Learning goals

- Distinguish primitives from objects reliably (including `null`).
- Encode the `typeof null` wart explicitly in `typeLabel`.

## What you will implement

Complete every `// TODO(Day17-task01):` in `src/primitivesLab.js` so `npm test` passes.

### `isPrimitiveValue(value)`

Return `true` iff `value` is a primitive:

- `null`, `undefined`, string, number, boolean, bigint, symbol

Return `false` for objects, arrays, and functions.

### `typeLabel(value)`

Return a stable string label:

- `null` → `"null"`
- arrays → `"array"` (use `Array.isArray`)
- functions → `"function"`
- plain objects → `"object"` (not null, not array, not function)
- otherwise return `typeof value` as a string
