# Day 17 — JavaScript Values, Types, and Variables

## Learning objectives

- Explain JavaScript’s **dynamic typing** model and how `typeof` behaves (including the `null` historic quirk).
- Choose between **`const`**, **`let`**, and legacy **`var`** with predictable scoping and reassignment rules.
- Use **template literals** safely for readable string assembly (including multiline output).
- Recognize common **coercion** footguns and write small guards for predictable user-facing conversions.

## Prerequisites

- HTML/CSS days **01–16** (you will read markup examples, but tasks are JS-only). Node **18+** for Vitest.

## How to use this folder

1. Read `DAY-17-THEORY-AND-REFERENCE.md`.
2. Each task is **JavaScript + Vitest**: from the task folder run `npm install` then `npm test`.
