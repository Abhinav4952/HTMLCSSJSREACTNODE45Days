# Day 14 — Flexbox Layout

## Learning objectives

- Explain **main axis vs cross axis** and how `flex-direction` flips them.
- Use **`justify-content`**, **`align-items`**, **`align-self`**, and **`gap`** deliberately (not randomly).
- Control wrapping with **`flex-wrap`** and **`order`** where appropriate.
- Apply **`flex` shorthand** (`grow shrink basis`) for dashboards—and know why **`min-width: 0`** prevents common overflow bugs.

## Prerequisites

- Days **07–13** (box model + responsive first pass). If media queries feel shaky, skim Day 13 again before Task 04.

## How to use this folder

1. Read `DAY-14-THEORY-AND-REFERENCE.md`.
