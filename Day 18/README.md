# Day 18 — Expressions, Operators, and Control Flow

## Learning objectives

- Write predictable **conditionals** and translate requirements into `if / else` without redundant branches.
- Choose the right loop construct (`for`, `while`, `for..of`) for a given collection task.
- Use **`switch`** for categorical dispatch while avoiding accidental fall-through bugs.
- Explain **truthiness** vs **strict equality** and encode strict checks where needed.

## Prerequisites

- **Day 17** (values, variables, coercion guards).
