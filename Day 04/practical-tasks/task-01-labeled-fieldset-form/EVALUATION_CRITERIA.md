# Evaluation Criteria — Day 04 — Task 01 — Labeled Fieldset Form

## Scope

Graded: HTML semantics for grouping + labels.

## Weighting (100 points)

| Category | Points | Description |
|----------|-------:|-------------|
| Correctness | 45 | Valid form; radios grouped; labels |
| Requirements coverage | 35 | Fieldsets + legend + submit |
| Code quality | 20 | Clear names (`name` attributes meaningful) |
| UX / a11y | 10 | Keyboard tab order sensible |

## Pass / fail gates

- Any control missing accessible name (visible label or `aria-label` with instructor approval—default: visible label).
- Radio options with different `name` values (cannot choose one ticket).

## AI grading prompt

Verify form post action placeholder, two fieldsets with legends, radio group same name, text+email labeled, submit button.
