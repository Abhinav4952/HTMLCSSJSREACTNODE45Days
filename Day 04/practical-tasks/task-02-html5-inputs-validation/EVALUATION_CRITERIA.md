# Evaluation Criteria — Day 04 — Task 02 — HTML5 Inputs Validation

## Scope

Graded: correct types and constraints; help text for pattern.

## Weighting (100 points)

| Category | Points | Description |
|----------|-------:|-------------|
| Correctness | 50 | Types/attributes match spec |
| Requirements coverage | 30 | Hint paragraph present and accurate |
| Code quality | 20 | Labels + names |
| UX / a11y | 10 | Errors understandable |

## Pass / fail gates

- Missing `pattern` or wrong pattern vs instructions.
- No explanatory help near `username`.

## AI grading prompt

Verify url/tel/number min/max, username pattern ^[a-z0-9_]{3,15}$ and visible hint text, all labeled.
