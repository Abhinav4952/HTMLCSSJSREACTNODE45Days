# Task 04 — GET vs POST Method Lab

## Summary

Create **two separate forms** on one page:

1. **Search** form: `method="get"`, `action="#"`, a single search field `name="q"`, submit **Search**.
2. **Issue report** form: `method="post"`, `action="#"`, short summary + details, submit **Report**.

Add a short `<section>` explaining in **your own words** when you would choose GET vs POST (≥60 words).

## Learning goals

- See query string vs body conceptually (with `action="#"`, browser may show `?q=` in URL bar for GET—observe DevTools Network on a real server later).
