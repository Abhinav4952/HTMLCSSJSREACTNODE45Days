# Task 03 — Textarea, Select, and Button Types

## Summary

Build a **feedback** form: `<textarea>` with `maxlength` and live-remaining counter is **not** required (no JS); instead, show `maxlength` in help text. Add `<select>` for category and three buttons: submit, reset, and a `button type="button"` that does nothing yet but is labeled **Preview** (no JS—click will no-op).

## Learning goals

- Differentiate reset vs submit vs neutral button.
- Use `<option>` values meaningfully.
