# Day 04 — Forms Foundation

## Learning objectives

- Build accessible forms: **every** control has an associated **label** (visible or explicitly wired with `for`/`id`).
- Use `<fieldset>` + `<legend>` to group related controls when appropriate.
- Choose correct **`type`** on `<input>` and basic validation attributes (`required`, `minlength`, `pattern` where taught).
- Use `<textarea>`, `<select>`, and `<button>` with correct **`type`** (`submit`, `button`, `reset`).
- Explain **GET vs POST** at a conceptual level and pick a method appropriate to the scenario.
