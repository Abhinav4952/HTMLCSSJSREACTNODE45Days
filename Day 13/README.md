# Day 13 — Responsive Design (First Pass)

## Learning objectives

- Build **fluid** layouts using flexible widths, `max-width`, and readable line lengths.
- Use **`min-width` / `max-width` media queries** with a deliberate mobile vs desktop strategy (mobile-first by default).
- Identify **breakpoint footguns** (padding added only at large widths, “orphan” rules, over-nesting).
- Combine **responsive images** (`max-width: 100%`, `height: auto`) with card grids that reflow.

## Prerequisites

- Days **07–12** (box model, positioning context, semantics). Review **Day 09** display patterns if media queries feel disconnected from layout.
