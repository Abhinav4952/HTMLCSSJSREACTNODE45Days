# Day 19 — Objects: Creation, Access, Methods, and Descriptors (Intro)

## Learning objectives

- Create and update **plain objects** predictably (shapes, keys, nested paths).
- Choose between **dot** vs **bracket** notation and dynamic keys.
- Attach **methods** to objects while keeping `this` expectations clear (plain objects only today).
- Read **property descriptors** with `Object.getOwnPropertyDescriptor` as a debugging/introspection tool.

## Prerequisites

- **Day 18** (control flow) and **Day 17** (types/coercion).
