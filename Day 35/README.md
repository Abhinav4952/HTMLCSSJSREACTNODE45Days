# Day 35 — React State & Lifecycle (Hooks + Class Framing)

## Learning objectives

- Use **`useState`** to store UI state in function components and trigger re-renders.
- Describe the **mount / update / unmount** mental model for React UI.
- Implement **`useEffect`** for side effects with correct **cleanup** where needed.
- Contrast **`setState` in class components** with hooks state updates (Day 35 emphasis; Day 38 revisits lifecycle framing).

## Prerequisites

- **Day 34** (components & props).
- Solid **JavaScript closures** review from Days 21–26 helps with effect dependencies.
