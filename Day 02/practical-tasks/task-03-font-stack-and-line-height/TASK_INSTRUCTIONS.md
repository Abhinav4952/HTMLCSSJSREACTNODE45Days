# Task 03 — Font Stack and Line Height

## Summary

Tune **typography in CSS** for a two-column reading layout (simple: `main` + `aside` or two `section`s). You will set `font-family` stacks, `font-size`, and `line-height` for `body` and headings.

## Learning goals

- Build a serif or sans stack with at least **three** font families in the list.
- Use **unitless** `line-height` on `body` and sensible `line-height` on headings.

## Provided files

- `index.html`, `styles/main.css`

## What you will implement

1. Complete HTML TODOs with short original paragraphs (≥120 words total).
2. Complete CSS TODOs for stacks and line-heights.

## Constraints

- Do not import webfonts or external stylesheets.
