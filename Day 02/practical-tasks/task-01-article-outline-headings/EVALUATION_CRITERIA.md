# Evaluation Criteria — Day 02 — Task 01 — Article Outline With Headings

## Scope

Graded: `index.html` heading hierarchy and article structure; `styles/main.css` optional minor edits only if TODO allows (default: HTML only).

## Weighting (100 points)

| Category | Points | Description |
|----------|-------:|-------------|
| Correctness | 40 | Valid HTML5; logical heading order |
| Requirements coverage | 35 | All TODOs; word count band |
| Code quality | 15 | Clean structure; no skipped levels |
| Readability | 10 | Writing quality |

## Pass / fail gates

- Skipped heading level (e.g. `h1` then `h3` without `h2` between globally in article).
- Lorem ipsum only.
