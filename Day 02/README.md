# Day 02 — Text Semantics & Formatting

## Learning objectives

- Choose semantic HTML text elements (`<strong>`, `<em>`, `<mark>`, `<abbr>`, `<code>`, `<kbd>`, `<samp>`) instead of generic `<span>` when meaning matches.
- Structure long readings with headings, paragraphs, lists, and block quotations.
- Apply CSS typography: `font-family` stacks, `font-size`, `font-weight`, `line-height`, and basic `color`.
- Predict **inheritance**: which properties flow from `body` to descendants by default.
