# Day 34 — React Components & Props

## Learning objectives

- Explain how **function components** map props to UI and why **props are read-only**.
- Compare **function vs class components** at a high level and know when each appears in legacy codebases.
- Use **JSX** with the automatic runtime (no per-file `import React` required when configured correctly).
- Validate props in development with **`prop-types`** and document expected shapes.
- **Compose** small presentational components into a simple screen layout.

## Prerequisites

- **Days 17–33** (modern JavaScript, including modules and classes).
- Comfort with **HTML semantics** and **CSS** from earlier weeks (you will style lightly inside JSX).

## How to use this folder

1. Read `DAY-34-THEORY-AND-REFERENCE.md` first.
2. Complete `practical-tasks/task-01-…` through `task-04-…` in order unless your instructor directs otherwise.
3. Each task is a standalone **Vite + React 18** app: from the task folder run `npm install`, then `npm run dev`. Use `npm run build` when you want a production bundle check.
