# Day 36 — Composition, `children`, and Lifting State

## Learning objectives

- Use **`props.children`** (and explicit element props) to build reusable layout shells.
- **Decompose** UI into presentational pieces with clear boundaries.
- **Lift state** to the nearest common ancestor when two siblings must stay in sync.
