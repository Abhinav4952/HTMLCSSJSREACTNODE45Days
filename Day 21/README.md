# Day 21 — Functions & Scope Basics

## Learning objectives

- Distinguish **function declarations**, **named function expressions**, and **anonymous** expressions—and know when each is appropriate.
- Use **arrow functions** for concise callbacks while recognizing their **lexical `this`** behavior (preview for Day 24).
- Apply **default parameters** with correct `undefined`-only semantics and avoid accidental coupling to mutable defaults.
- Read **hoisting** and the **temporal dead zone (TDZ)** as execution-order rules, not magic—especially for `var` vs `let`/`const`.

## Prerequisites

- **Days 17–20** (values, control flow, objects, arrays).
