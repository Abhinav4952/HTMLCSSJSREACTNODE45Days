# Day 16 — Advanced Responsive Layout & Typography

## Learning objectives

- Compare **mobile-first vs desktop-first** media-query strategies and document breakpoints with rationale (not random widths).
- Implement **responsive typography** using `clamp()`, `calc()`, and fluid line-length constraints together.
- Perform a **practical page revision**: take an imperfect responsive page and fix overflow, type scale, and breakpoint behavior without rewriting unrelated content.
- Use **`@container` queries** where they reduce coupling between a component and the viewport (with a documented fallback).

## Prerequisites

- Days **07–15** (layout, flex, first-pass responsive, transitions). If media queries feel rusty, skim **Day 13** before Task 01.

## How to use this folder

1. Read `DAY-16-THEORY-AND-REFERENCE.md`.
2. Complete tasks in order. Task 03 is the largest **integration** pass; Task 04 introduces **container queries** (challenge if your target browser set is older—see task notes).
