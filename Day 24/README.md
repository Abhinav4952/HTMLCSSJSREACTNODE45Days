# Day 24 — Dynamic `this`, `call`, `apply`, and `bind`

## Learning objectives

- Predict **dynamic `this`** for simple method calls vs **standalone** function calls.
- Use **`call`** and **`apply`** to invoke a function with an explicit receiver and arguments.
- Build **partial applications** with **`bind`** without losing arity clarity.
- Recognize **strict-mode** `this` footguns and fix them with `bind` or arrows.

## Prerequisites

- **Days 21–23** (functions through coercion comparisons).
